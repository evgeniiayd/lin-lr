## Задача оптимизации производства электроники («ТехноПром»)


Проект решает задачу линейного программирования для максимизации прибыли компании при ограниченных ресурсах.


## Структура проекта

```
technoprom_optimization/
├── main.py                  # Основное решение задачи
├── visualization.py         # Визуализация результатов
├── data/
│   └── problem_data.json  # Входные данные задачи
├── requirements.txt         # Зависимости
└── README.md              # Документация
```

## Описание файлов

- **main.py** — реализует математическую модель, решает задачу оптимизации с помощью `scipy.optimize.linprog`, выводит оптимальное решение и анализ использования ресурсов.
- **visualization.py** — строит график допустимой области, ограничений и оптимального решения
- **data/problem_data.json** — содержит входные данные (затраты ресурсов, запасы, прибыль)
- **requirements.txt** — список необходимых библиотек

## Как запустить проект

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите основное решение:
   ```bash
   python main.py
   ```

3. Запустите визуализацию:
   ```bash
   python visualization.py
   ```

## Требования

- Python 3.8+
- numpy
- scipy
- matplotlib
- json (встроен в Python)

## Результаты работы

Программа выводит:
- оптимальное количество смартфонов и планшетов для производства;
- максимальную прибыль;
- анализ использования ресурсов (процент использования каждого ресурса).

Визуализация показывает:
- допустимую область решений;
- линии ограничений по ресурсам;
- оптимальное решение (точка);
- линии уровня целевой функции (изопрофиты).
